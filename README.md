# WikiQuote Scraper Bot

The program's idea is simply to access a website specialized in quotes and phrases (the site https://ar.wikiquote.org was used). 

The program analyzes the website's HTML code and then navigates to multiple windows on the site based on the user's request, retrieving a random phrase on a specific topic also according to the user's request.

## Features
* It can extract more than one phrase at the same time.
* The user has the freedom to choose the number of desired quotes and phrases.

> **Note:** The program needs updating based on changes to the website, but the general idea has not changed.
